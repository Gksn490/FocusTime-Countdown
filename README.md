# FocusTime - Countdown & Goal Timer

FocusTime is a Flutter mobile app that helps you create and track countdowns for events and personal goals. It features real-time countdown updates, a modern gradient-themed UI, optional notes and emojis for each countdown, and the ability to organize and filter events by categories like Travel, Birthday, or Goals.

## Features
- Real-time countdown updates
- Gradient-themed, modern user interface
- Add notes and emojis to each countdown
- Organize countdowns by categories
- Filter countdowns by category

## Installation
To install and run the app, first clone the repository using `git clone <YOUR_REPO_URL>`, then navigate to the project folder, install the required dependencies with `flutter pub get`, and finally launch the app by running `flutter run`.
